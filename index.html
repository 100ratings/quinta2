Quick Start Guide

What this trainer is

This tool helps you master QUINTA mechanics: forcing the 2nd from the left among 5 items by deciding the correct Counting Type and Starting Side for any number N.

How to use (2-minute flow)

  1. Pick a Range (or set a Custom min/max). Leave Adaptive on.
  2. Press Generate number (N) or type in Enter N and click Find side & type.
  3. Choose Type: even ‚Üí Basic, odd ‚Üí Bounce.
  4. Choose Side via the mod-8 rule.
  5. (Optional) Toggle Step-by-step to visualize.
  6. Hit Check (Enter). If correct, a new number appears automatically; if wrong, revise and tente novamente.
  7. Use Reset training to clear stats and weights.

Manual mode

Type any N and click Find side & type (or Ctrl+Enter).
